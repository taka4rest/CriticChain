# Contributing to CriticChain

Thank you for your interest in contributing to CriticChain!

## Getting Started

1. Fork this repository
2. Clone your fork and create a new branch
3. Install dependencies: `pip install -r requirements.txt`
4. Copy `.env.example` to `.env` and add your Gemini API key
5. Run the app: `streamlit run app.py`

## How to Contribute

- **Bug Reports**: Open an issue with steps to reproduce
- **Feature Requests**: Open an issue describing your use case
- **Pull Requests**: For non-trivial changes, please open an issue first to discuss

## Pull Request Guidelines

- Keep PRs focused on a single change
- Include a clear description of what and why
- Ensure the app runs without errors before submitting

## Code Style

- Python 3.10+
- Follow existing patterns in the codebase
- User-facing strings must be in English only
